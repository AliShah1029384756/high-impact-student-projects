# 05. Databases and SQL

A good developer understands both application logic and data design.

## Core Topics

- Relational modeling
- Keys and constraints
- Joins and aggregations
- Indexing fundamentals
- Transactions and ACID

## SQL Practice Plan

- Week 1: SELECT, WHERE, ORDER BY, LIMIT
- Week 2: JOINs, GROUP BY, HAVING
- Week 3: Subqueries and window functions
- Week 4: Indexes, optimization basics

## Portfolio-Ready DB Projects

- Expense tracker with category analytics
- Task manager with status and due-date filters
- Student attendance and marks dashboard

## Common Mistakes

- No schema planning before coding
- Storing everything as text
- Ignoring indexes on frequently queried fields
