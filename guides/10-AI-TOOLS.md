# 10. Free and Student-Friendly AI Tools for Coding

AI tools can speed up learning when used responsibly.

## Free or Free-Tier Tools

- ChatGPT: https://chatgpt.com/
- Google Gemini: https://gemini.google.com/
- Claude: https://claude.ai/
- Codeium: https://codeium.com/
- Tabnine: https://www.tabnine.com/

## Student Programs and Benefits

- GitHub Education Pack: https://education.github.com/pack
- GitHub Copilot for students (via Education): https://education.github.com/pack
- JetBrains student licenses: https://www.jetbrains.com/community/education/
- Notion for education: https://www.notion.so/product/notion-for-education

## How To Choose a Tool

- For explanations: ChatGPT, Claude
- For code completion in IDE: Copilot, Codeium, Tabnine
- For brainstorming architecture: Gemini, ChatGPT

## Pitfalls to Avoid

- Copy-pasting without understanding
- Ignoring security and data privacy
- Relying on one output without verification
