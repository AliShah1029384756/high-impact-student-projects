# 02. Programming Practice

Consistency beats intensity. Practice with pattern tracking, not random problem solving.

## Best Platforms

- LeetCode: https://leetcode.com/
- HackerRank: https://www.hackerrank.com/
- Codeforces: https://codeforces.com/
- AtCoder: https://atcoder.jp/
- Exercism: https://exercism.org/
- Codewars: https://www.codewars.com/

## Practice Structure

### Weekly Plan
- 3 easy problems
- 2 medium problems
- 1 review session

### Monthly Plan
- Re-solve 10 old problems without hints
- Write one pattern summary document

## Pattern Categories to Cover

- Two pointers
- Sliding window
- Binary search
- Prefix sums
- DFS and BFS
- Dynamic programming

## Tracking Template

- Problem link
- Pattern used
- Initial idea
- Final approach
- Mistakes made
- Complexity analysis

## Recommended Repositories

- LeetCode patterns: https://github.com/seanprashad/leetcode-patterns
- TheAlgorithms: https://github.com/TheAlgorithms
- Coding Interview University: https://github.com/jwasham/coding-interview-university

## Anti-Burnout Tips

- Stop after 90 minutes if stuck
- Learn one pattern deeply instead of five poorly
- Focus on explanation quality, not only accepted submissions
