# 04. Data Structures and Algorithms

DSA is not just interview prep. It improves problem-solving quality in all software projects.

## What To Master

- Arrays and strings
- Linked lists
- Stacks and queues
- Trees and binary search trees
- Graphs
- Hash maps and sets
- Heaps and priority queues

## Algorithm Areas

- Sorting and searching
- Recursion and backtracking
- Dynamic programming
- Greedy algorithms
- Graph traversals

## Best Resources

- NeetCode roadmap: https://neetcode.io/roadmap
- LeetCode: https://leetcode.com/
- GeeksforGeeks DSA: https://www.geeksforgeeks.org/data-structures/
- MIT OCW algorithms: https://ocw.mit.edu/courses/6-006-introduction-to-algorithms-fall-2011/

## Study Method

1. Learn concept
2. Solve 2-3 guided problems
3. Solve 5 independent problems
4. Explain solution out loud
5. Write pattern notes

## Interview Readiness Standard

- Medium problems solved consistently
- Able to derive complexity quickly
- Can explain trade-offs in approach
