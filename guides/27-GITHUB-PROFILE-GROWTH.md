# GitHub Profile Growth Guide

Use this guide to present your work clearly for recruiters, mentors, and collaborators.

## Profile Essentials

- Clear profile bio with target role
- Pinned repositories aligned with your role
- Consistent contribution history
- Strong repository READMEs

## Pinned Repo Strategy

Pin 4-6 repositories:
1. Best full-stack project
2. Best frontend or backend project
3. DSA or interview-prep repository
4. Open source contribution evidence
5. One documentation-heavy project

## README Quality Checklist

- Problem statement
- Features list
- Tech stack
- Setup instructions
- Screenshots or demo
- Learning outcomes

## Monthly Improvement Sprint

1. Improve one old project README
2. Close one pending issue
3. Add one quality contribution
4. Publish one project update
