# 30-60-90 Day Skill Plans

This guide gives practical execution plans by phase so progress stays measurable.

## 30-Day Phase (Foundation)

Weekly outcomes:
1. Week 1: Basics and setup
2. Week 2: Core implementation practice
3. Week 3: Mini project
4. Week 4: Publish and reflect

Deliverables:
- 1 mini project
- 1 clean README
- 1 learning log

## 60-Day Phase (Depth)

Weekly outcomes:
1. Weeks 5-6: Intermediate concepts
2. Weeks 7-8: Real project module

Deliverables:
- 1 medium project with auth/data flow
- 1 deployment
- 1 case-study writeup

## 90-Day Phase (Career-Ready)

Weekly outcomes:
1. Weeks 9-10: Interview-focused practice
2. Weeks 11-12: Portfolio polishing and outreach

Deliverables:
- 2 polished projects
- Updated resume and profile
- Internship/job application tracker

## Tracking Template

Use this weekly template:
- Goal of week
- Skills practiced
- Tasks completed
- Blockers
- Next-week fix
