# Real-World Project Idea Bank

Use this guide when you want projects that look practical to recruiters and clients.

## How To Select a Project

1. Choose one problem you understand.
2. Keep MVP scope finishable in 2-4 weeks.
3. Add one differentiator: analytics, automation, accessibility, or local-language support.

## Beginner Projects

1. Smart To-Do with priorities and deadlines
2. Expense tracker with monthly insights
3. Student notes app with tag-based search

## Intermediate Projects

1. University deadline aggregator
2. Internship readiness tracker
3. Bus schedule optimizer with alerts

## Advanced Projects

1. Complaint intelligence dashboard
2. Team contribution auditor
3. Scholarship eligibility recommender

## Portfolio Impact Score (Simple)

Score each project out of 10:
- Real problem relevance: 0-2
- Technical depth: 0-2
- UI and usability quality: 0-2
- Documentation quality: 0-2
- Deployment and demo quality: 0-2

## Project Card Template

- Problem
- Users
- MVP features
- Tech stack
- Demo link
- Lessons learned
