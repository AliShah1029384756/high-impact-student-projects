# Internship and Job Hunt Kit

This guide is a practical system for applying consistently with better quality.

## Weekly Application Cycle

1. Monday: shortlist roles
2. Tuesday: tailor resume and profile
3. Wednesday: send applications
4. Thursday: networking messages
5. Friday: interview prep and follow-up

## Core Assets You Need

- One-page role-specific resume
- Updated profile README and pinned projects
- Portfolio with 2-4 strong projects
- Professional outreach template

## Cold Outreach Template (Short)

Use a 4-part structure:
1. Who you are
2. Why this company/role
3. What relevant work you have done
4. Clear ask (internship consideration or short call)

## Application Tracker Columns

- Company
- Role
- Date applied
- Status
- Follow-up date
- Notes

## Interview Follow-Up Template

Send within 24 hours:
- Thank interviewer
- Mention one specific discussion point
- Reconfirm interest
- Offer any additional material
