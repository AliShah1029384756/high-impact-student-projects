This guide organizes interview preparation into practical packs by topic.

## Pack 1: DSA Basics

Focus:
- Arrays, strings, hashing, two pointers
- Recursion, stacks, queues, linked lists

Practice target:
- 30-40 quality problems

## Pack 2: SQL and Data

Focus:
- Joins, group by, window functions basics
- Query optimization fundamentals

Practice target:
- 20 SQL questions with explanation

## Pack 3: Backend and API

Focus:
- HTTP methods and status codes
- Auth flow and secure APIs
- Caching and pagination basics

Practice target:
- 15 system/API questions

## Pack 4: Frontend and UI

Focus:
- Rendering, state, hooks lifecycle
- Accessibility and performance basics

Practice target:
- 15 role-focused frontend questions

## Pack 5: Behavioral Questions

Structure:
- Situation
- Task
- Action
- Result

Prepare 6-8 real stories from projects, teamwork, and deadlines.

## Mock Interview Workflow

1. 20-minute technical round
2. 15-minute project deep dive
3. 10-minute behavioral round
4. Review and improvement notes
