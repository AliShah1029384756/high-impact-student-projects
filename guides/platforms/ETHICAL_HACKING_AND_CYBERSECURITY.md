# Ethical Hacking and Cybersecurity Platforms (Legal Learning)

Use only legal labs and authorized practice environments.

## Beginner-Friendly Platforms

- Cisco Skills for All (Cybersecurity): https://skillsforall.com/
- TryHackMe: https://tryhackme.com/
- PortSwigger Web Security Academy: https://portswigger.net/web-security
- OWASP Top 10: https://owasp.org/www-project-top-ten/

## Intermediate Platforms

- Hack The Box Academy: https://academy.hackthebox.com/
- CyberDefenders: https://cyberdefenders.org/
- PicoCTF: https://picoctf.org/
- OverTheWire Wargames: https://overthewire.org/wargames/

## Certification-Oriented Routes

- Google Cybersecurity Professional Certificate: https://www.coursera.org/professional-certificates/google-cybersecurity
- ISC2 Certified in Cybersecurity (program info): https://www.isc2.org/certifications/cc
- CompTIA Security+ overview: https://www.comptia.org/certifications/security

## Suggested 8-Week Simple Plan

- Weeks 1-2: Basics (networking, OWASP Top 10)
- Weeks 3-4: Web labs (PortSwigger + TryHackMe)
- Weeks 5-6: SOC/forensics beginner labs
- Weeks 7-8: Certification prep + project write-up

## Portfolio Proof Ideas

- Web vulnerability lab write-ups (legal targets only)
- OWASP Top 10 summary notes
- Security checklist for one demo web app

## Safety Rules

- Do not scan or test systems without permission.
- Do not share exploit steps for unauthorized use.
- Keep all learning in legal lab environments.
