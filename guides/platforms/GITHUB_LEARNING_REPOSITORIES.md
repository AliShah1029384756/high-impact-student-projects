- OSSU Computer Science: https://github.com/ossu/computer-science

- System Design Primer: https://github.com/donnemartin/system-design-primer

- TheAlgorithms: https://github.com/TheAlgorithms
- Build Your Own X: https://github.com/codecrafters-io/build-your-own-x
- Awesome Lists: https://github.com/sindresorhus/awesome
- Project-Based Learning: https://github.com/practical-tutorials/project-based-learning

## How To Study GitHub Repositories Effectively

1. Read README fully before cloning.
2. Use Issues and Discussions to understand common confusion points.
3. Implement one section yourself, then compare.
4. Keep learning notes in your own repo.

## Contribution Guide

- Improve docs first if you are a beginner.
- Keep pull requests small and focused.
- Follow repository contribution rules.
