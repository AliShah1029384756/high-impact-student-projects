# MOOCs and Structured Course Platforms

## High-Quality Platforms

- CS50 (Harvard): https://cs50.harvard.edu/x/
- edX: https://www.edx.org/
- Coursera: https://www.coursera.org/
- Udacity: https://www.udacity.com/
- FutureLearn: https://www.futurelearn.com/
- MIT OCW: https://ocw.mit.edu/

## Free or Financial Aid Options

- Coursera Financial Aid: https://www.coursera.org/financial-aid
- edX support and assistance: https://support.edx.org/hc/en-us/articles/215167857
- GitHub Education Pack: https://education.github.com/pack

## Suggested Path by Goal

- Beginner: CS50 + freeCodeCamp + one project track
- Intermediate: Full Stack Open + SQL + API course
- Advanced: system design + cloud + one specialization

## Completion Strategy

- Focus on one course at a time
- Finish assignments before starting next module
- Publish outcomes in GitHub README
