## Verification and Sharing Checklist

- Save completion screenshot and certificate link.
- Add one project as proof of applied skill.
- Add certificate section in GitHub README.
- Mention completion month and key outcomes.

## Simple Rule

Certificates are strongest when paired with projects. Always publish proof work with each certificate.
