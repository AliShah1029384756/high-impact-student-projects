# 16. Scholarships, Grants, and Student Benefits

A practical resource list for students looking for funding and education support.

## International Scholarship Sources

- DAAD scholarship database: https://www.daad.de/en/studying-in-germany/scholarships/
- Erasmus+ opportunities: https://erasmus-plus.ec.europa.eu/
- Fulbright programs: https://foreign.fulbrightonline.org/

## Pakistan-Focused Scholarship and Funding Sources

- HEC Scholarships Portal: https://scholarships.hec.gov.pk/
- Ehsaas Undergraduate Scholarship (HEC): https://www.hec.gov.pk/english/scholarshipsgrants/ehsas/Pages/default.aspx
- PEEF Scholarships: https://peef.org.pk/
- Punjab Educational Endowment Fund details: https://www.peef.org.pk/cmms/
- NAVTTC skill scholarships: https://navttc.gov.pk/

## MOOC Financial Aid

- Coursera financial aid: https://www.coursera.org/financial-aid
- edX support and assistance: https://support.edx.org/hc/en-us/articles/215167857

- freeCodeCamp certifications: https://www.freecodecamp.org/learn/
- Cisco Skills for All badges: https://skillsforall.com/
- Postman Student Expert badge: https://studentexpert.postman.com/
- Microsoft Learn achievements: https://learn.microsoft.com/training/
- Google Cloud Skills Boost: https://www.cloudskillsboost.google/

## Application Tips

- Keep academic transcripts and recommendation drafts ready
- Prepare one strong statement of purpose template
- Track deadlines in a dedicated calendar
- Verify each opportunity through official domains

## Simple Document Kit (Keep Ready)

- Updated CV (1 page + detailed version)
- Latest transcript and student ID copy
- 2 recommendation templates (faculty + professional)
- Statement of purpose master draft
- GitHub portfolio links list
