# 19. Free Courses and Certifications Hub (Student Edition)

Practical list of free or scholarship-friendly courses for students who want portfolio value, verified skill proof, and job-ready outcomes.

## How To Use This Guide

1. Pick one domain only (do not start all at once).
2. Complete one beginner + one intermediate course.
3. Build one project after each course.
4. Add certificate links and project outcomes in your GitHub README.

## Upload Checklist (Before Sharing Publicly)

- Verify links are active.

- Do not include private chat/contact links.
- Add outcomes and project proof, not only course names.
