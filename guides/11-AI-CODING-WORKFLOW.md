# 11. AI Coding Workflow That Helps Learning

Use AI as a mentor and reviewer, not a replacement engineer.

## Recommended Workflow

1. Define your feature and constraints.
2. Ask AI for implementation plan and edge cases.
3. Write first draft yourself.
4. Ask AI for code review and potential bugs.
5. Add tests and documentation.

## Practical Prompt Sequence

- Prompt 1: "Give implementation plan for feature X"
- Prompt 2: "Review this draft for edge cases"
- Prompt 3: "Suggest minimal refactor for readability"
- Prompt 4: "Generate test cases for success and failure scenarios"

## Validation Checklist

- [ ] I understand every line before committing
- [ ] I tested expected and unexpected input
- [ ] I documented trade-offs in README
- [ ] I can explain my final design in interview format
