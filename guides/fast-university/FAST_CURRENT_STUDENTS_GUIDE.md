# Guide for Current FAST Students

## Academic Success Framework

### Semester Strategy

- First 2 weeks: finalize course plan and deadlines
- Weekly: lecture revision + assignment split
- Mid semester: performance review and backlog recovery
- Final month: test-focused revision schedule

## Skills To Build Alongside Degree

- Git and GitHub workflow
- Documentation and technical writing
- Project architecture basics
- Interview problem-solving patterns

## Practical Student Stack

- Notes: Notion or Obsidian
- Tasks: Trello or GitHub Projects
- API testing: Postman
- Diagrams: Excalidraw

## Teamwork and FYP Readiness

- Track tasks clearly per member
- Hold weekly updates
- Keep contribution transparency
- Store all major decisions in versioned docs

## Well-Being and Consistency

- Avoid all-nighters as default routine
- Keep realistic weekly study targets
- Use focused deep-work blocks
