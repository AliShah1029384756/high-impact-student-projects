# FAST Resource Link Verification Guide

This guide ensures shared links are accurate, current, and genuinely helpful.

## Verification Checklist

- [ ] Link opens without login wall (or clearly marked if login required)
- [ ] Source is official or trusted community publication
- [ ] Content is educational and relevant to FAST applicants/students/graduates
- [ ] Link is active in the current semester/year
- [ ] No private or restricted material is being redistributed

## Source Priority

1. Official university sources (admissions, notices, departments)
2. Official platform documentation
3. Public educational repositories with active maintenance
4. High-quality community content with clear attribution

## How To Validate Before Adding

1. Open the link in an incognito/private browser window.
2. Confirm content title and publisher match the claim.
3. Record a short note: why this link is useful.
4. Add a `Last verified` date near the link list.

## Link Quality Labels

Use these tags with shared links:
- `official` for university/company source
- `community-verified` for trusted public community resources
- `archival` for older but still useful references

## Review Cadence

- Re-check high-priority links every 30 days.
- Re-check all links at least once per semester.

## Removal Rules

Remove links that are:
- broken for more than 14 days
- misleading or outdated
- posted without proper source attribution
