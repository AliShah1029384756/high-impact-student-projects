# Guide for FAST Graduates

## First 6 Months After Graduation

1. Polish portfolio and GitHub profile
2. Finalize resume and LinkedIn
3. Practice DSA and interview communication
4. Build one production-quality project

## Career Tracks

- Software engineering
- Data and AI
- DevOps and cloud
- Product and analytics

## What Recruiters Look For

- Project quality and clarity
- Ability to explain decisions
- Team and communication skills
- Consistency in growth

## Graduate Action Checklist

- [ ] 6 polished repositories
- [ ] 1 deployed full-stack project
- [ ] interview practice log
- [ ] resume tailored for target roles
- [ ] references and recommendations ready
