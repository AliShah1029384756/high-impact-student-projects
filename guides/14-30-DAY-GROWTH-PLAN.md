# 14. 30-Day Practical Growth Plan

A focused monthly plan for measurable progress.

## Week 1: Foundations Refresh

- Revise HTML, CSS, and JavaScript basics
- Build one mini frontend project
- Solve 5 easy coding problems

## Week 2: DSA Consistency

- Study two core patterns (e.g., two pointers, sliding window)
- Solve 10 problems
- Write one pattern summary note

## Week 3: API and Backend

- Build one API-based project
- Add input validation and error handling
- Write project README with setup and usage

## Week 4: Polish and Publish

- Improve documentation across repos
- Add tests where possible
- Deploy one project and share demo link

## Daily Time Split (Example)

- 60 minutes coding
- 30 minutes DSA
- 20 minutes docs and reflection

## End-of-Month Output

- 1 published project
- 1 improved old project
- 1 updated profile README section
