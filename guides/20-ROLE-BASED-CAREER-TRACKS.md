Use this guide to choose one focused career direction instead of learning everything at once.

## How To Use

1. Pick one role for 8-12 weeks.
2. Follow the required skills list.
3. Build the portfolio projects for that role.
4. Use the readiness checklist before applying.

## Track 1: Frontend Developer

Core skills:
- HTML, CSS, JavaScript fundamentals
- React and component architecture
- State management and API integration
- Responsive design and accessibility

Portfolio projects:
- Dashboard with charts and filters

## Role Selection Rule

- If you enjoy UI and interactions: start with Frontend.
- If you like logic and systems: start with Backend.
- If you prefer data insights: start with Data Analyst.
- If you like defense and risk analysis: start with Cybersecurity.
- If you enjoy quality and edge cases: start with QA.
