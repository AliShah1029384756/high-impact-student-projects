# AI for Students (Practical Use)

This guide focuses on practical, ethical AI usage for learning and productivity.

## High-Value Use Cases

1. Topic summaries from long notes
2. Quiz generation for active recall
3. Code explanation and debugging hints
4. Presentation draft support
5. Interview question simulation

## Prompt Patterns

- Explain like I am a beginner, with one real-world example.
- Convert this topic into 10 exam-style MCQs.
- Review this code and list logic issues only.
- Turn these notes into a 5-slide presentation outline.

## Safe Usage Rules

- Verify facts from trusted sources
- Never submit AI output without review
- Use AI as assistant, not replacement
- Keep private data out of prompts

## Learning Workflow

1. Learn concept manually
2. Use AI for clarification
3. Build mini implementation
4. Reflect and document learning
