# 01. CS Foundations (Start Here)

A strong foundation prevents confusion later when projects become complex.

## What To Learn First

- Programming fundamentals
- Data structures and algorithms basics
- Computer systems basics
- Database fundamentals
- Networking essentials

## Recommended Resources (Working Links)

- OSSU Computer Science: https://github.com/ossu/computer-science
- CS50 by Harvard: https://cs50.harvard.edu/x/
- MIT OpenCourseWare: https://ocw.mit.edu/
- Teach Yourself CS: https://teachyourselfcs.com/
- Stanford Online (CS playlists): https://www.youtube.com/@stanfordonline

## Foundation Checklist

- [ ] I can write clean code in one language (JS or Python)
- [ ] I can explain arrays, linked lists, stacks, queues, trees, and hash maps
- [ ] I understand time and space complexity basics
- [ ] I can design a simple relational schema
- [ ] I understand client-server basics and HTTP methods

## Suggested 8-Week Foundation Plan

### Weeks 1-2
- Language basics, loops, functions, and data structures
- Practice 10 easy coding problems

### Weeks 3-4
- OOP basics and modular coding
- Build one mini project from scratch

### Weeks 5-6
- Intro to databases and SQL
- Build one CRUD app with storage

### Weeks 7-8
- Intro to computer networks and APIs
- Document all learning in GitHub notes

## Common Mistakes

- Jumping to frameworks too early
- Memorizing without building
- Ignoring documentation skills

## Output You Should Have

- 1 language confidence
- 1 mini project
- 1 CRUD app
- 1 clean learning note repository
