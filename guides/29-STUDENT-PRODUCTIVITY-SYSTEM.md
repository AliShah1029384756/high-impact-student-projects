# Student Productivity System

Use this simple operating system to stay consistent without burnout.

## Daily Structure

- Deep work block 1: learning
- Deep work block 2: project building
- Light block: revision or admin tasks

## Weekly Review Template

1. What worked
2. What failed
3. Why it failed
4. One fix for next week

## Focus Rules

- One major priority per day
- No multitasking during deep work
- Keep notifications off during study
- Use short breaks intentionally

## Burnout Prevention Basics

- Keep one rest block weekly
- Avoid excessive all-nighters
- Keep realistic target scope
- Track sleep and stress patterns
