# 07. DevOps, Cloud, and Deployment

Deployment and automation skills make student projects look professional.

## Core Skills

- CI/CD basics
- Containerization
- Environment variables and secrets
- Logging and monitoring

## Learning Resources

- Docker docs: https://docs.docker.com/
- Kubernetes docs: https://kubernetes.io/docs/home/
- GitHub Actions docs: https://docs.github.com/actions
- Vercel docs: https://vercel.com/docs
- Netlify docs: https://docs.netlify.com/
- Render docs: https://render.com/docs

## Minimum Deployment Checklist

- [ ] App runs in production mode
- [ ] Environment variables configured
- [ ] Error logging enabled
- [ ] README includes deploy link
- [ ] Health endpoint added for backend

## Student-Friendly Deployment Stack

- Frontend: Vercel or Netlify
- Backend: Render or Railway
- Database: Supabase, Neon, or MongoDB Atlas

## CI Starter Pipeline

- Install dependencies
- Run lint
- Run tests
- Build
- Deploy on main branch
