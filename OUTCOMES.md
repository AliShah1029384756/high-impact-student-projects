# Outcomes (30/60/90 Days)

Measure real progress, not only reading completion.

## 30-Day Outcomes

- Complete one learning path from START_HERE
- Finish at least one project with README
- Solve at least 20 coding problems
- Build one weekly routine and follow it

## 60-Day Outcomes

- Publish 2 to 3 practical projects
- Improve one older project with better docs
- Demonstrate API integration or backend basics
- Write one career-oriented project case note

## 90-Day Outcomes

- Maintain 4 to 6 professional repositories
- Have one deployed and documented end-to-end project
- Be interview-ready on core DSA patterns
- Explain system design basics for one project

## Self-Scoring Rubric (1-5)

- Consistency
- Code quality
- Documentation quality
- Problem-solving depth
- Interview readiness

Target score:
- 30 days: >= 3 average
- 60 days: >= 3.5 average
- 90 days: >= 4 average
