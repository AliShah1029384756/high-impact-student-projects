# FAQ

## 1. Where should I start?
Start with [START_HERE.md](START_HERE.md) and choose one path only.

## 2. How many hours should I study daily?
1.5 to 3 focused hours are enough if done consistently.

## 3. Beginner vs intermediate - how do I decide?
If you cannot build a basic CRUD app independently, follow the beginner path first.

## 4. How should I use AI ethically?
Read [guides/13-AI-ETHICS.md](guides/13-AI-ETHICS.md) and never submit code you do not understand.

## 5. Which projects should I build first?
Use beginner order: Calculator -> Todo App -> Notes App.

## 6. Do I need all guides?
No. Use [GUIDE_SELECTION_MATRIX.md](GUIDE_SELECTION_MATRIX.md) for goal-based reading.

## 7. How do I prepare for internships?
Use [guides/04-DSA.md](guides/04-DSA.md), [guides/06-SYSTEM-DESIGN.md](guides/06-SYSTEM-DESIGN.md), and [guides/09-CAREER-RESUME-INTERVIEW.md](guides/09-CAREER-RESUME-INTERVIEW.md).

## 8. How do I know a resource link is reliable?
Follow [guides/fast-university/FAST_LINK_VERIFICATION_GUIDE.md](guides/fast-university/FAST_LINK_VERIFICATION_GUIDE.md).

## 9. Can I contribute resources?
Yes. Read [CONTRIBUTING.md](CONTRIBUTING.md) and use the submission template.

## 10. How do I track progress?
Use [OUTCOMES.md](OUTCOMES.md), [LAST_VERIFIED.md](LAST_VERIFIED.md), and your monthly notes.
