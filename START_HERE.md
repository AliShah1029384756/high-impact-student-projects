# Start Here

## Path 1: Beginner

Use this if you are new to CS or coding.

1. Read [guides/01-CS-FOUNDATIONS.md](guides/01-CS-FOUNDATIONS.md)
2. Follow [guides/03-WEB-DEVELOPMENT.md](guides/03-WEB-DEVELOPMENT.md)
3. Execute [guides/14-30-DAY-GROWTH-PLAN.md](guides/14-30-DAY-GROWTH-PLAN.md)
4. Use [FAQ.md](FAQ.md) when stuck

- You can build and document beginner projects independently.

## Path 2: FAST Student

Use this if you are applying to FAST, currently studying, or planning your next step.

1. Open [guides/fast-university/FAST_UNIVERSITY_INDEX.md](guides/fast-university/FAST_UNIVERSITY_INDEX.md)
2. Pick your stage guide: applicants, current students, or graduates
3. Verify shared links with [guides/fast-university/FAST_LINK_VERIFICATION_GUIDE.md](guides/fast-university/FAST_LINK_VERIFICATION_GUIDE.md)
4. Submit useful resources through [guides/fast-university/FAST_RESOURCE_SUBMISSION_TEMPLATE.md](guides/fast-university/FAST_RESOURCE_SUBMISSION_TEMPLATE.md)

Expected outcome:
- You follow a focused, stage-specific plan with reliable references.
